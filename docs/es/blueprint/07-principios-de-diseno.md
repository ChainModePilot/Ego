# Principios de Diseño

1. **Listo para Usar** — Los usuarios comunes pueden comenzar con tres comandos después de `pip install`. No se requiere conocimiento de machine learning ni configuración de drivers de GPU.
2. **Completamente Automatizado** — Desde la recolección de datos hasta la salida del modelo, sin intervención manual necesaria. Como una lavadora: metes la ropa, presionas inicio, sacas la ropa limpia.
3. **Adaptación Multi-Versión** — Una ejecución de entrenamiento produce múltiples versiones para diferentes capacidades de cómputo. El mismo "tú" puede vivir simultáneamente en un reloj inteligente, teléfono y computadora.
4. **Ejecución Repetible** — Reentrena después de actualizar los datos, y Ego evoluciona en consecuencia. A medida que tú creces, Ego crece también.
5. **Base Intercambiable** — Cambia de modelo base cuando se lanzan nuevos, reentrena LoRA. Como actualizar el procesador de un teléfono — el rendimiento mejora pero la "personalidad" permanece igual.
6. **Extensible por Plugins** — Los adaptadores de fuentes de datos, modelos base y ediciones son todos extensibles. La comunidad puede contribuir nuevos plugins de fuentes de datos (ej., adaptador de Notion, importador de historial de chat de WhatsApp).
7. **Privacidad Primero** — Todo el entrenamiento puede completarse localmente, los datos nunca salen del dispositivo. Tu diario, historial de chat y notas de trabajo permanecen para siempre en tu propia computadora.
