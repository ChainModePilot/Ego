# 路线图

## Phase 1：最小可用（MVP）
- [x] 支持本地目录数据源
- [x] 支持单一基座（Qwen 3.5 0.8B）
- [x] 支持单一 edition（normal）
- [x] 基础语料构建（对话格式）
- [x] QLoRA 训练管线
- [x] CLI 工具（ego init / train / run）

## Phase 2：多版本支持
- [x] 四个默认 edition（lite/normal/plus/super）
- [x] 多基座支持（Qwen、Gemma、Phi）
- [x] 自定义 edition
- [x] 版本管理与回滚

## Phase 3：数据源扩展
- [x] URL 数据源支持
- [ ] 云存储数据源（S3/OSS）
- [x] 数据源插件机制
- [x] 增量数据采集与增量训练

## Phase 4：智能化
- [ ] 自动评估对齐度
- [ ] 训练参数自动调优
- [ ] 定时自动训练（调度器）
- [ ] Web UI 管理界面

## Phase 5：iFay 集成
- [ ] iFay 数据格式适配器
- [ ] Fayger 加载协议对接
- [ ] Ego 包分发机制
