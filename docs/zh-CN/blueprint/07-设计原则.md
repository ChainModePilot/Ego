# 设计原则

1. **开箱即用** — 普通人 `pip install` 后三条命令就能用。不需要懂机器学习，不需要配置 GPU 驱动。
2. **全自动化** — 从数据采集到模型产出，无需人工干预。就像洗衣机：放进衣服，按下开始，拿出干净衣服。
3. **多版本适配** — 一次训练，产出适配不同算力终端的多个版本。同一个"你"，可以同时住在手表、手机和电脑里。
4. **可重复执行** — 数据更新后重新训练，Ego 随之进化。你在成长，Ego 也在成长。
5. **基座可替换** — 新模型发布后可切换基座，重新训练 LoRA。就像给手机换了更好的处理器，性能提升但"性格"不变。
6. **插件可扩展** — 数据源适配器、基座模型、edition 均可扩展。社区可以贡献新的数据源插件（如 Notion 适配器、微信聊天记录导入器）。
7. **隐私优先** — 所有训练可在本地完成，数据不出设备。你的日记、聊天记录、工作笔记，永远只在你自己的电脑上。
