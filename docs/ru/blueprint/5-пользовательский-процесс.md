# Пользовательский процесс

## Для обычных пользователей

```
Шаг 1: Установка Ego Pod
  $ pip install ego-pod

Шаг 2: Инициализация конфигурации
  $ ego init
  → Интерактивный помощник: выбор базовой модели, настройка путей к источникам данных, выбор редакций
  → Как настройка нового телефона — просто следуйте подсказкам шаг за шагом

Шаг 3: Запуск обучения
  $ ego train
  → Автосбор данных → сборка корпуса → обучение LoRA → мульти-версионный выпуск Ego
  → За время, пока вы пьёте кофе, ваш персонализированный ИИ готов

Шаг 4: Использование Ego
  $ ego run --edition normal
  → Загрузка базы + LoRA, запуск ИИ-модели, настроенной под вас
  → Попробуйте пообщаться с ней, посмотрите, «звучит ли она как вы»

Шаг 5: Обновление Ego (после изменения данных)
  $ ego train --update
  → Инкрементальный сбор данных → переобучение → перезапись старой версии
  → Узнали что-то новое? Перековка — и Ego догоняет
```

## Интеграция с iFay

Тело iFay сообщает Ego адреса источников данных через конфигурационные файлы. Ego Pod автоматически завершает обучение, а произведённые LoRA-адаптеры загружаются Fayger.

> Например: Ваш iFay работает на телефоне, периодически синхронизируя новые данные в указанный каталог, а затем запуская переобучение Ego Pod. После завершения обучения новый LoRA-адаптер автоматически отправляется на ваш телефон, и Ego «эволюционирует» — весь процесс для вас невидим.
