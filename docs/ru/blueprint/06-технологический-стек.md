# Технологический стек

| Компонент | Технология | Обоснование |
|-----------|-----------|-------------|
| Язык программирования | Python | Наиболее зрелая экосистема AI/ML |
| Фреймворк обучения | Unsloth / PEFT / MLX | Мульти-бэкенд автоадаптация, покрывающая NVIDIA GPU, Apple Silicon, CPU |
| Метод обучения | QLoRA (PEFT) | 4-битная квантизация, обучение на одном GPU, дружественно к потребительскому оборудованию |
| Формат модели | SafeTensors | Безопасный (без риска pickle), быстрая загрузка, стандарт HuggingFace |
| Среда выполнения инференса | llama.cpp / Ollama | Кроссплатформенность, лёгкость, поддержка квантизированных моделей |
| Формат конфигурации | YAML | Человекочитаемый, легко редактируемый |
| Управление пакетами | pip / conda | Стандартная экосистема Python |
