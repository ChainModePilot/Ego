# 프로젝트 개요

Ego는 iFay의 "두뇌"——호스트와 고도로 정렬된 개인화 소형 모델입니다.

동시에, Ego는 독립적으로 사용 가능한 오픈소스 도구이기도 합니다: **일반인을 위한, 바로 사용 가능한 개인화 모델 훈련 도구**. 누구나 자신의 데이터를 "연성"하여 자신을 이해하는 AI 모델을 만들 수 있습니다.

> 예를 들어: 당신이 사진 애호가로, 촬영 노트, 보정 선호도, 장비 선택 기록을 대량으로 축적했다고 합시다. Ego Pod를 통해 이 데이터들은 당신의 미적 스타일을 이해하는 AI 모델로 "연성"됩니다——따뜻한 색조를 선호하고, 35mm 화각을 좋아하며, 후보정 시 화이트밸런스를 먼저 조정한 후 톤커브를 당기는 습관을 알고 있습니다. 이 모델은 스마트폰에서 실행되어 언제든 개인화된 촬영 조언을 제공합니다.

## 두 가지 역할

| 역할 | 설명 | 예시 |
|------|------|------|
| iFay의 두뇌 | iFay 바디가 제공하는 데이터 소스 주소를 수신하고, Sense/Action/Thought/Skill/Credentials 데이터를 학습하여 호스트의 사고 패턴과 행동 습관을 내재화 | 당신의 iFay가 스마트폰에서 실행되고, Ego가 일정 습관을 학습한 후 매일 아침 자동으로 당일 일정을 계획합니다. 스타일은 당신이 직접 계획하는 것과 완전히 동일 |
| 독립 도구 | iFay 없이도 실행 가능하며, 개인 전용 Agent 모델을 빠르게 훈련하는 범용 도구로 기능 | 고객 서비스 팀이 Ego Pod로 베테랑 상담원의 대화 기록을 모델로 연성하여, 신입이 베테랑의 커뮤니케이션 스타일과 문제 처리 방식을 학습할 수 있게 함 |
