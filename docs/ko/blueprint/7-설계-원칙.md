# 설계 원칙

1. **즉시 사용 가능** — 일반인이 `pip install` 후 세 개의 명령어로 사용 가능. 머신러닝 지식도 GPU 드라이버 설정도 불필요.
2. **완전 자동화** — 데이터 수집부터 모델 출력까지, 수동 개입 불필요. 세탁기처럼: 옷을 넣고, 시작을 누르고, 깨끗한 옷을 꺼내기.
3. **다중 버전 적응** — 한 번의 훈련으로, 다양한 컴퓨팅 파워의 단말에 적합한 여러 버전 출력. 같은 "당신"이 스마트워치, 스마트폰, PC에 동시에 존재 가능.
4. **반복 실행 가능** — 데이터 업데이트 후 재훈련하면, Ego도 함께 진화. 당신이 성장하면, Ego도 성장.
5. **베이스 교체 가능** — 새 모델 출시 후 베이스를 전환하고 LoRA 재훈련. 스마트폰 프로세서를 업그레이드하듯, 성능은 향상되지만 "성격"은 변하지 않음.
6. **플러그인 확장 가능** — 데이터 소스 어댑터, 베이스 모델, edition 모두 확장 가능. 커뮤니티가 새로운 데이터 소스 플러그인 (Notion 어댑터, 카카오톡 채팅 기록 임포터 등) 기여 가능.
7. **프라이버시 우선** — 모든 훈련을 로컬에서 완료 가능, 데이터가 기기 밖으로 나가지 않음. 일기, 채팅 기록, 업무 노트는 영원히 당신의 컴퓨터에만 존재.
