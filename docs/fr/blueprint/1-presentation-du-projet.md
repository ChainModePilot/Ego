# Présentation du projet

Ego est le « cerveau » d'iFay — un petit modèle personnalisé hautement aligné avec son hôte.

En même temps, Ego est aussi un outil open source autonome : **un outil d'entraînement de modèle personnalisé prêt à l'emploi pour le grand public**. N'importe qui peut l'utiliser pour « forger » ses propres données en un modèle d'IA qui le comprend.

> Par exemple : Vous êtes un passionné de photographie qui a accumulé de nombreuses notes de prise de vue, préférences de retouche et historiques de choix de matériel. Grâce à Ego Pod, ces données sont « forgées » en un modèle d'IA qui comprend votre style esthétique — il sait que vous préférez les tons chauds, que vous privilégiez la focale 35 mm, et que vous ajustez habituellement la balance des blancs avant de tirer les courbes en post-traitement. Ce modèle peut fonctionner sur votre téléphone, vous fournissant des conseils de prise de vue personnalisés à tout moment.

## Deux identités

| Identité | Description | Exemple |
|----------|-------------|---------|
| Cerveau d'iFay | Reçoit les adresses de sources de données fournies par le corps iFay, apprend les données Sense/Action/Thought/Skill/Credentials, et intériorise les schémas de pensée et les habitudes comportementales de l'hôte | Votre iFay fonctionne sur votre téléphone. Après qu'Ego a appris vos habitudes de planification, il organise automatiquement votre journée chaque matin exactement dans le même style que vous le feriez |
| Outil autonome | Peut fonctionner indépendamment d'iFay, servant d'outil généraliste pour entraîner rapidement des modèles d'Agent personnels | Une équipe de service client utilise Ego Pod pour forger les historiques de conversation d'agents expérimentés en un modèle, permettant aux nouveaux venus d'apprendre le style de communication et les approches de résolution de problèmes des vétérans |
