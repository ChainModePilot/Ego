# 使用者使用流程

## 面向一般人的使用方式

```
第一步：安裝 Ego Pod
  $ pip install ego-pod

第二步：初始化設定
  $ ego init
  → 互動式引導：選擇基座模型、設定資料來源路徑、選擇 edition
  → 就像設定一支新手機，按提示一步步來就行

第三步：開始訓練
  $ ego train
  → 自動採集資料 → 建構語料 → 訓練 LoRA → 輸出多版本 Ego
  → 喝杯咖啡的時間，你的個人化 AI 就煉好了

第四步：使用 Ego
  $ ego run --edition normal
  → 載入基座 + LoRA，啟動一個與你對齊的 AI 模型
  → 試著跟它聊天，看看它是不是「像你」

第五步：更新 Ego（資料變化後）
  $ ego train --update
  → 增量採集新資料 → 重新訓練 → 覆蓋舊版本
  → 你最近學了新東西？重新煉一次，Ego 就跟上了
```

## 面向 iFay 的整合方式

iFay 軀殼透過設定檔告訴 Ego 資料來源位址，Ego Pod 自動完成訓練，產出的 LoRA 適配器供 Fayger 載入。

> 例如：你的 iFay 跑在手機上，它會定期把你的新資料同步到指定目錄，然後觸發 Ego Pod 重新訓練。訓練完成後，新的 LoRA 適配器自動推送到你的手機，Ego 就「進化」了——整個過程你完全無感。
