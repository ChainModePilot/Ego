# 技術棧

| 元件 | 技術選型 | 理由 |
|------|---------|------|
| 程式語言 | Python | AI/ML 生態最成熟 |
| 訓練框架 | Unsloth / PEFT / MLX | 多後端自動適配，涵蓋 NVIDIA GPU、Apple Silicon、CPU |
| 訓練方法 | QLoRA (PEFT) | 4-bit 量化，單卡即可訓練，消費級硬體友善 |
| 模型格式 | SafeTensors | 安全（無 pickle 風險）、快速載入、HuggingFace 標準 |
| 推論執行環境 | llama.cpp / Ollama | 跨平台、輕量、支援量化模型 |
| 設定格式 | YAML | 人類可讀，易於編輯 |
| 套件管理 | pip / conda | Python 標準生態 |
