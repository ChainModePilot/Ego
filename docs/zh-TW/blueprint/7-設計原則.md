# 設計原則

1. **開箱即用** — 一般人 `pip install` 後三條指令就能用。不需要懂機器學習，不需要設定 GPU 驅動。
2. **全自動化** — 從資料採集到模型產出，無需人工介入。就像洗衣機：放進衣服，按下開始，拿出乾淨衣服。
3. **多版本適配** — 一次訓練，產出適配不同算力終端的多個版本。同一個「你」，可以同時住在手錶、手機和電腦裡。
4. **可重複執行** — 資料更新後重新訓練，Ego 隨之進化。你在成長，Ego 也在成長。
5. **基座可替換** — 新模型發布後可切換基座，重新訓練 LoRA。就像給手機換了更好的處理器，效能提升但「個性」不變。
6. **外掛可擴充** — 資料來源適配器、基座模型、edition 均可擴充。社群可以貢獻新的資料來源外掛（如 Notion 適配器、LINE 聊天紀錄匯入器）。
7. **隱私優先** — 所有訓練可在本機完成，資料不出裝置。你的日記、聊天紀錄、工作筆記，永遠只在你自己的電腦上。
