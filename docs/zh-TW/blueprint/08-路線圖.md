# 路線圖

## Phase 1：最小可用（MVP）
- [x] 支援本機目錄資料來源
- [x] 支援單一基座（Qwen 3.5 0.8B）
- [x] 支援單一 edition（normal）
- [x] 基礎語料建構（對話格式）
- [x] QLoRA 訓練管線
- [x] CLI 工具（ego init / train / run）

## Phase 2：多版本支援
- [x] 四個預設 edition（lite/normal/plus/super）
- [x] 多基座支援（Qwen、Gemma、Phi）
- [x] 自訂 edition
- [x] 版本管理與回溯

- [x] 增量資料採集與增量訓練

## Phase 4：智慧化
- [ ] 自動評估對齊度
- [ ] 訓練參數自動調優
